# SwanAPI

快速的机器学习项目变云端/本地端API工具。

**预期功能：**

1. 深度学习镜像一站式打包
   - 只需编写简单的配置文件和推理文件
   - 无需烦恼PyTorch、TensorFlow、Transformers等主流的机器学习库的安装
   - 无需烦恼CUDA、cudnn等GPU环境配置
   - 镜像运行时将运行一个高性能API服务，轻松调用
2. 推送镜像实现云托管
3. 调用[SwanHub](https://swanhub.co/)上的云端API
4. 在本地生成调试GUI界面与API文档
